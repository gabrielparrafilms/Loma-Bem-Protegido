Glassmorphism

.glass-card utility

The Glass Card

The signature glassmorphism effect is built using custom CSS to ensure high-fidelity rendering. It combines an ultra-low opacity white background, severe background blurring, multiple inner shadows to create the edge highlights, and an absolute positioned highlight layer for the soft glare.

.glass-card {
  background: rgba(255, 255, 255, 0.03);
  backdrop-filter: blur(20px);
  box-shadow: 0 30px 60px -15px rgba(0, 0, 0, 0.8),
    inset 0 1px 1px rgba(255, 255, 255, 0.15);
  border-radius: 1.5rem;
}
